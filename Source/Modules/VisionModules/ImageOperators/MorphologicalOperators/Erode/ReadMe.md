<br><br>

## Outputs

|Name|Description|
|:----|:-----------|

|Name|Description|Type|Default value|
|:----|:-----------|:----|:-------------|

## Long description
Module that applies the 3x3 erode operator to an image (here generalized as
        the max of the pixels in the area).
